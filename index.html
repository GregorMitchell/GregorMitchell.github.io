DRIVR - An at Home Golf Swing Trainer

For my Honours Project I decided to create an at home golf swing analyser using Unity and Arduino. The system uses an IR sensor and a Gyroscope to measure the speed and angle of the user's swing, which are then transmitted to Unity using Bluetooth serial ports. The golf ball flight is then calculated and simulated to the user using the Unity game engine.

Outbreak - Zombie Shooter With a Twist

For our Games Programming module, we were tasked with creating a game of our choosing. We were required to come up with an idea, plan and then implement the game using an engine of our choosing. I created Outbreak, a top-down zombie shooter that requires you to cure the zombies, as you play as a doctor sworn to their Hippocratic oath. The game features path-finding AI for the zombies, meaning they can chase you all around the map. I decided to use Godot as my engine for this game, as I had prior experience using it.
